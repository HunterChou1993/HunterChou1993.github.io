                                        # 11 组件化开发

                                        # 1. 组件分类

                                        1. 根据组件定义方式 函数组件和类组件

                                        2. 根据状态

                                        • 无状态组件和有状态组件

                                        3.根据职责

                                        • 展示型组件和容器型组件

                                        注意

                                        • 函数式组件 无状态组件 展示型组件 主要关心UI的展示
                                        • 类组件 有状态组件 容器型组件 主要关注数据逻辑

                                        # 2. 类组件

                                        # 2.1 要求

                                        • 组件名称是大写字符开头
                                        • 必须继承React.Comonent
                                        • 类组件必须实现render函数

                                        # 2. 组件内部

                                        • constructor 可选的，初始化数据
                                        • this.state 维护组件内部数据
                                        • render() 唯一必须实现的方法

                                        # 3. render函数

                                        它检查到this.props和this.state 发生变化的时候，会被调用 返回值

                                        • React元素 - 通过jsx创建出来的 - 无论是div还是组件，只要被jsx创建出来的都是
                                        • 数组或者fragments
                                        • Portals 可以渲染子节点到不同的DOM子树中
                                        • 字符串或者数值类型

                                        # 函数式组件

                                        使用function来定义的函数，只是这个函数会返回和类组件中render函数返回一样的内容

                                        # 使用

                                        不需要继承 要有返回值

                                        # 接收数据

                                        放在props里面

                                        function AppFunction() {
    return <h1>App Functional</h1>
}

export default AppFunction

                                        # 类组件和函数式组件区别

                                        • 函数式组件没有生命周期
                                        • 函数式组件没有this
                                        • 没有内部的状态
